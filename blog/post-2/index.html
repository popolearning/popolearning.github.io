一、為什麼不用WordPress架站？

一直都有想架設個人網站的念頭，之前也嘗試購買了虛擬空間搭配WordPress架設了網站，雖然有很多外掛可以使用，但總覺得很多限制，選主題也選了老半天，都挑不到期待的樣式，有一些功能想增加，也無法如預期的加在希望的位置上(可能是我功力不足，看起來有放置 html 程式碼的地方，但我就是用不好)，如果想要能夠更自由的調整，也需要使用付費的主題，但也依然有其限制。

加上空間的費用，雖然不是很多，當時一次買了4年，1年算下來約台幣594元，再加上網域名稱，922元/年，網址還比空間貴，完全忘記當初怎麼挑的。

其實如果有賺錢的狀態下，這樣的費用是可以負擔，但偏偏我都是在沒工作的時候才有空學新東西，也才有空做網站，這樣固定的支出就變得吃力，而且後來發現，可能是空間便宜的關係，連線的速度實在有點難耐，有時候會很慢，種種都讓我對於用wordpress架站到底是不是長久之計打了問號。

不用WordPress的原因：費用高、找不到滿意的主題與功能




二、推薦這堂課的理由

推薦指數：★★★★★

課程本身便宜，老師也很有誠意，教學仔細邏輯清楚，基本上會遇到的問題課程都有提到，另外一個很大的好處就是學了Github架站就能夠免費架設個人網站 ! 成功放上網路的時候真的覺得很有成就感！

主要是教大家用 Github + Hugo 架設個人網站，撰寫文章用的markdown語法也有做清楚的介紹，基本上跟著課程一步一步來，就可以順利架好網站。

課程中介紹Hugo的主題可以在官網上挑選，但有的看起來很強大的，像是還可以切換白天夜晚模式，偏華麗的那種沒有想像中容易安裝，雖然他的下載頁面會有安裝的說明，但還是沒有那麼容易成功，我也是試了好一陣子，才挑到現在這個主題。

可以選擇跟老師用一樣的主題做練習，熟悉做法之後再去找喜歡的主題，我當初是到40+ Best Completely Free Hugo Themes For 2024找到想要的，並且可以成功運行的主題。

也可以在google搜尋"hugo template free"還有很多其他的可以挑。




四、課程優點

  • 老師講解清楚仔細
  • 價格親民
  • 老師回覆問題認真
  • 真的可以成功架設個人網站
  • 適合新手



五、課程缺點

  • 不是每個主題都能用課程教的方法套用，會有些變化，需要一些程式能力去操作
  • 整體算是新手友善，要再進階的功能就不包含了



六、這堂課適合誰？

我本身有稍微學過 html + CSS，以及一些 Python 跟 javascript，算是知道大概架構，所以可以去推斷網站該從哪邊去修改程式碼。
因此學習起來覺得很清楚，很有成效，如果是完全的程式新手，其實老師講得非常仔細，新手的話我認為也可以聽懂，但在想要增加進階功能的時候，可能會比較苦惱，但是感謝偉大的科技，我們可以使用ChatGPT來協助寫程式，也可以問它要在哪邊改，或怎麼改，所以其實搭配ChatGPT使用應該是人人都能上手。

只要是想架個人網站，又不想支付太多費用，不害怕程式，並且有耐心的人，都很適合喔！




七、上完課後我的收穫？

  • 實現架設個人網站的夢想
  • 學會 Github + Hugo 這個架站方式
  • 調整程式碼的過程中，激發求知慾，覺得很有趣



八、結論

最後總結一下，我認為這堂課890元很值得，雖然看似初階，但真的完全實現老師寫的課程目標，讓大家能夠跟著架設網站，當然製作的過程中，會需要一些程式碼的基礎能力，所以如果能夠搭配學習Html+css系列課程，會更得心應手。但即便是新手，也可以使用ChatGPT來輔助完成，所以真的很推薦喔！
